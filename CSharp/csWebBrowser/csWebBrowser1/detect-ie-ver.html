[CSS-Table, IE8+] If the color blocks are displayed vertically, the engine is IE7 or older.
[CSS Transform, IE9+] If the color blocks are not shrunk, the engine is IE8 or older.
[Flexbox, IE10+] If the color blocks are displayed vertically, the engine is IE9 or older.
